# Coffee Shop Management System

The Coffee Shop Management System is a **desktop-based application** designed to efficiently manage the daily operations of a coffee shop.  
The system supports **Admins and Employees**, enabling streamlined order processing, inventory tracking, menu management, invoice generation, and financial monitoring through an intuitive graphical interface.
